Li8Z - CHIP-8 Emulator

A CHIP-8 emulator written in the Zig programming language. Features include:

  • Full CHIP-8 instruction set implementation
  • Support for both desktop and web platforms
  • Audio and keyboard input support
  • Built with Zig, Raylib, and WebAssembly

Brainoof - Esoteric Language Interpreter

An interpreter for an esoteric programming language, written in pure Rust, inspired by Krzysztof Gabis's C implementation. Features include:

  • Pure Rust implementation
  • Capable of running complex programs like bottles.bf
  • Includes a Python implementation as bonus
  • Simple and efficient command-line interface
